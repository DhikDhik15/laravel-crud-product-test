# 📈 Aplikasi Pendataan Pegawai

API Untuk manajemen produk.

---

## 💻 Tech Spec
- Laravel 12
- PHP 8.3
- SQL Lite

## ⚙️ Instalasi

1. Clone repository:
    ```bash
    git clone <url>
    cd employee-app
    ```
2. composer install
3. cp .env.example .env
4. php artisan key:generate
5. php artisan migrate
6. php artisan db:seed
7. php artisan serve
